# FundedCore — Breach Guardian (Chrome extension)

An always-on HUD that shows your **Distance to Breach** on top of any trading
platform, on any page. It reads your account equity, applies your firm's exact
drawdown + daily-loss rules, and tells you how much room you have left — and the
largest size you can take right now without blowing the account.

## Install (load unpacked, ~30 seconds)
1. Open **chrome://extensions** in Chrome (or Edge: edge://extensions).
2. Turn on **Developer mode** (top-right toggle).
3. Click **Load unpacked** and select this `extension` folder.
4. Pin "FundedCore — Breach Guardian" and click it to configure: pick your firm,
   set how it reads your equity (Manual / Auto-detect / CSS selector), Save.

## Try the demo
Open `demo.html` (in this folder) — or the live site's demo page — set the HUD to
**Auto-detect**, and drag the equity slider. Watch the HUD react in real time.

> Illustrative rule snapshots. Always confirm specifics with your firm.
