# FundedCore — The AI-Native Prop Firm

Rebuilt from the ground up around a single idea: **stop selling challenges, start underwriting edge.**
Connect a trading history → get a verified, transparent **Trader Score** → get funded directly.
No challenge fee, no $500 upfront, keep up to 90% of profits.

## What's inside
- **`lib/score.ts`** — the Trader Score Engine. A real actuarial model that turns a list of trades
  into four sub-scores (Edge Quality, Consistency, Discipline, Drawdown Control), a composite
  0–100 Trader Score with a confidence factor, a grade, a funding decision, and a capital offer.
  Fully transparent — every metric that moved the number is surfaced.
- **`lib/csv.ts`** — a flexible broker/journal CSV parser (auto-detects pnl/date/size/symbol/side).
- **`lib/sampleTraders.ts`** — four seeded demo traders that each land on a different verdict.
- **`app/page.tsx`** — landing / pitch (the $400M middleman, the flywheel, the economics).
- **`app/apply/page.tsx`** — the funding application: upload a CSV or pick a sample, scored live.
- **`app/dashboard/page.tsx`** — the funded-trader dashboard.
- **`components/`** — hand-rolled SVG visuals (score gauge, radar, equity curve), zero chart deps.

## Run locally
```bash
npm install
npm run dev      # http://localhost:3000  (set VERCEL=1 to use the Vercel/SSR config)
```
